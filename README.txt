Thank you for downloading!

This mod changes the color of the Polish tanks from their default brown to a semi-historical gray-green color.
Included are skins for all Polish vehicles and matching icons for the garage and tech tree.

To install:
- Drop the "CWW-RecolorPack-Polish-Szarozielony-Gray-Green" folder into <Your WoT Directory>\mods\<game version number>. Example: C:\Games\World_Of_Tanks\mods\1.0.2.4.

Enjoy!
